## What was done this session (2026-03-22, v10 session)

### Campaign v1.13 вАФ level intro screens
Level intro screens shown before each level on every entry (both from level select
and after completing a level). `LEVEL_INTROS` data array (8 entries) and
`LevelIntroScreen` component added. Router gained `'levelIntro'` screen state.

### Campaign v1.14 вАФ time attack star rating bug fix
Stars were evaluated before pending tile falls resolved. `pendingTimeExpiry` state
flag defers `setGameState` until `isAnimating === false && combo === 0`.

### Campaign v1.15 вАФ transition screen fields + campaign complete screen
Transition screen now shows: moves used (non-TA), best combo, specials created,
bonus moves earned, bonus move pool total, campaign running total. Campaign complete
screen added after L8: total score, stars/40, per-level breakdown, "Play Again"
(keep records) and "Reset all and start again" (clears data, starts at L1 intro).

### Campaign v1.16 вАФ game over screen improvements
Personal best score + star rating shown in game over overlay. Score formatted.
"Start over from Level 1 (keeps records)" link added. `CampaignGame` receives
`highScore`, `bestStars`, `onStartOver` props from `Match3Campaign`.

### Campaign v1.17 вАФ text fixes
L1 intro heading: "Welcome to Campaign!" вЖТ "Welcome!". "colour" вЖТ "color".

### Campaign v1.18 вАФ campaign total score resets between runs
`campaignTotalScore` was summed from all-time high scores, carrying over between
runs. Fix: `runTotalScore` state in `Match3Campaign` accumulates per-win scores
for the current run only. Resets in `handleBackToMap`, `handleStartOver`,
`handleResetAll`.

### Campaign v1.19 вАФ run total resets when selecting L1 from map
`handleSelectLevel` now resets `runTotalScore` to 0 when `levelIndex === 0`.

### Campaign v1.20 вАФ remove рЯП¶ icon throughout
- Header counter: `рЯП¶ N` вЖТ `Bonus moves: N`
- Burst notification: `+1 рЯП¶` вЖТ `+1 bonus move` (14px text)
- L5 intro: `icon: 'рЯП¶'` вЖТ `icon: ''` (no icon rendered)
- L5 intro bullet updated to reference "Bonus moves" by name
- `LevelIntroScreen` skips icon div when `intro.icon` is falsy

### Campaign v1.21 вАФ L4 intro combo/cascade bullet
Replaced two separate cascade bullets with a single plain-language description:
"Combinations of matches (combos) and each match that occurs after (cascades)
create score multipliers that lead to big increases."
