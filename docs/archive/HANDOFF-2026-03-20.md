## localStorage keys

| Key | Type | Description |
|---|---|---|
| `match3_highScore` | integer string | All-time high score (legacy) |
| `match3_highCombo` | integer string | All-time high combo (legacy) |
| `match3_highTurnScore` | integer string | All-time high single-turn score (legacy) |
| `match3_stats` | JSON string | Full game stats + history (v11.2+) |
| `match3_bankedMoves` | integer string | Persistent banked move count (v11.2+) |

`match3_stats` schema (version 1):
```json
{
  "version": 1,
  "gamesPlayed": 0,
  "gamesWon": 0,
  "gamesLost": 0,
  "bonusRoundsTaken": 0,
  "earlyEnds": 0,
  "movesSaved": 0,
  "history": [
    {
      "ts": 1711234567890,
      "won": true,
      "finalScore": 8420,
      "levelTarget": 6500,
      "movesRemaining": 3,
      "endType": "bonusRound",
      "difficultyBonus": 1200,
      "maxCombo": 4
    }
  ]
}
```
History ring buffer: max 50 entries (oldest dropped when full).
