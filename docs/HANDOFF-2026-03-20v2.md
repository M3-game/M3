## Repo / build notes

- Vite multi-page project, `npm run build`
- Deploys to GitHub Pages via `.github/workflows/deploy.yml` on push to main
- Base path: `/M3/` — set in `vite.config.js`
- All version URLs: `docs/version-urls.md`
- Tests: `npm test` — Vitest, pure logic only (25 passing)
- Archive convention: move current file to `platforms/<platform>/archive/` before
  creating new versioned file. Never overwrite.
- Shared code: `core/` — `AdminPanel.jsx`, `gameLogic.js`, `gameLogic.test.js`
- Shared campaign logic: `levels/campaignConfig.js`
- Docs archive: `docs/archive/`

## localStorage keys

### Arcade
| Key | Type | Description |
|---|---|---|
| `match3_highScore` | integer string | All-time high score |
| `match3_highCombo` | integer string | All-time high combo |
| `match3_highTurnScore` | integer string | All-time high single-turn score |
| `match3_stats` | JSON string | Full game stats + history (v11.2+) |
| `match3_bankedMoves` | integer string | Persistent banked move count (v11.2+) |

### Campaign
| Key | Type | Description |
|---|---|---|
| `match3_campaign_level` | integer | Highest level reached |
| `match3_campaign_highScores` | JSON array | Best score per level |
| `match3_campaign_stars` | JSON array | Best stars per level |
| `match3_campaign_totalScore` | integer | Cumulative campaign score |
| `match3_campaign_bonusMoves` | integer | Banked campaign bonus moves |
